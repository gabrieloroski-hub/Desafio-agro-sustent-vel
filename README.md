Desafio Agro-Sustentável — Agrinho 2026
Este projeto consiste em uma plataforma educacional interativa desenvolvida especialmente para o Programa Agrinho 2026, concorrendo na categoria de projetos escolares. O trabalho foi totalmente guiado pelo tema oficial "Agro-forte, futuro sustentável: equilíbrio entre produção e meio ambiente", com a orientação da nossa escola e professores.

O principal objetivo do site é transformar dados técnicos e desafios complexos do agronegócio brasileiro em um conhecimento acessível, leve e dinâmico. A intenção é mostrar para a comunidade escolar e para a sociedade que a tecnologia é a chave para o "Agro-Forte" continuar produzindo alimentos em larga escala sem agredir a natureza, estimulando o pensamento crítico sobre o consumo e a produção consciente.

Na prática, o site funciona como um jogo de perguntas e respostas focado na realidade do campo. O usuário passa por um quiz dinâmico que aborda temas cruciais da atualidade, como a dependência e a otimização do uso de fertilizantes no Brasil, o combate ao desmatamento ilegal através do uso de inteligência no campo e as regras de preservação do Código Florestal. Durante a navegação, o usuário conta com uma barra de progresso visual e recebe feedback educativo instantâneo: ao clicar em uma opção, a plataforma destaca imediatamente a resposta correta, garantindo que o jogador aprenda com o erro ou fixe o acerto. No final, uma tela personalizada exibe a pontuação com mensagens motivacionais baseadas no desempenho.

Toda a construção técnica foi feita pensando na leveza e na modernidade. O site utiliza a tríade pura do desenvolvimento web: HTML5 para a estruturação semântica do conteúdo, CSS3 para criar uma identidade visual atraente baseada nas cores da terra e da natureza através de um layout totalmente responsivo para celulares e computadores, e JavaScript para controlar a lógica das perguntas, validação de acertos e mudança de telas. O código foi organizado de forma modular, separando a estrutura, o estilo e a lógica em arquivos independentes para facilitar futuras melhorias. Além disso, aplicamos conceitos fundamentais de programação, como manipulação de matrizes de dados, funções orientadas a eventos e controle de estados no navegador.

O projeto também se destaca pelo seu compromisso social e ambiental no ambiente digital. No quesito acessibilidade, a paleta de cores e o tamanho das fontes foram rigorosamente planejados para oferecer um alto contraste, garantindo uma leitura confortável e inclusiva para pessoas com baixa visão ou dificuldades de leitura. Pensando na sustentabilidade digital, o site foi desenvolvido no conceito de "Green Code" (Código Verde). Por ser um código puro, sem o uso de bibliotecas externas ou frameworks pesados, ele consome menos dados de internet e exige menos processamento dos servidores e dos celulares dos usuários. Isso reduz diretamente o consumo de energia elétrica dos aparelhos, diminuindo a pegada de carbono digital do projeto.

Para quem deseja testar o projeto no computador, o processo é extremamente simples. Basta baixar os arquivos da plataforma para uma pasta local e dar um duplo clique no arquivo principal de estrutura. O sistema abrirá de forma automática no navegador padrão da máquina, sem a necessidade de instalar qualquer programa ou conexão obrigatória com servidores externos.
